## 💪 製品概要
自分しか知らない穴場スポットを共有する掲示板サイト

### 🔥 背景(製品開発のきっかけ、課題等)
- 世の中にある良いんだけど知られていない穴場をもっと広めたい
- インターネット上に上がってきづらい価値ある情報を拾い上げたい

### 🚀 追加予定の機能
- Flashメッセージ
- いいね機能
- 保存機能
- 位置情報の紐付け
- 検索機能
